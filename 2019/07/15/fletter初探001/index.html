flutter常用组件

Text组件

负责显示文本和定义显示样式

Text常用属性

  1. textAlign：文本对齐方式，可用值有：
    • TextAlign.center，水平居中
    • TextAlign.left，水平居左
    • TextAlign.right，水平居右
    • TextAlign.start与TextAlign.left类似
    • TextAlign.end与TextAlign.right类似
  2. maxLines：最大行数配合overflow属性一起使用
  3. overflow：超出最大行数时的裁剪方式，可用值有：
    • TextOverflow.clip，直接裁剪
    • TextOverflow.ellipsis：超出三个点点点
    • TextOverflow.fade：从上到下淡入
  4. style: 文本样式可以定义文本的字体大小、颜色、粗细等
  5. data：数据为显示的文本
  6. textDirection：文本书写方向。TextDirection.ltr从左到右，TextDirection.rtl从右到左
  7. textScaleFactor：字体缩放系数
  8. textSpan：文本块

Container组件

容器组件（Container组件）包含一个子Widget

Container常用属性

  1. alignment：控制child的对齐方式，可用的值有：
    • Alignment.center
    • Alignment.bottomCenter
    • Alignment.bottomLeft
    • Alignment.bottomRight
    • Alignment.centerLeft
    • Alignment.centerRight
    • Alignment.topCenter
    • Alignment.topLeft
    • Alignment.topRight
  2. width：Container的宽度
  3. height：Container的高度
  4. color：设置Container的背景色
  5. padding：Decoration内部空白的区域
  6. margin：围绕在Decoration和child之外的空白区域，不属于内容区域
  7. decoration：绘制在child后面的装饰，设置了decoration就不能设置color，可以设置渐变色(gradient)，边框(border)，圆角(borderRadius)等

提示：padding与margin的不同之处在于，padding是包含在Content内，而margin是外部边界。设置点击事件，padding区域会响应，而margin区域不会响应

Image 组件

显示图片的组件

构造函数

  1. new Image：从ImageProvider获取图像
  2. new Image.asset：加载资源图片
  3. new Image.file：加载本地图片文件
  4. new Image.network：加载网络图片
  5. new Image.memory：加载Unit8List资源图片

Image常用属性

  1. width/height：设置图片容器的大小
  2. fit：图片填充模式，可用值
    • BoxFit.fill：全图显示，显示可能拉伸，充满
    • BoxFit.contain：全图显示，显示原比例，不需充满
    • BoxFit.cover：显示可能拉伸，可能裁剪，充满
    • BoxFit.fitWidth：显示可能拉伸，可能裁剪，宽度充满
    • BoxFit.fitHeight：显示可能拉伸，可能裁剪，高度充满
    • BoxFit.none：原始大小
    • BoxFit.scaleDown：与BoxFit.contain类似，但是不允许显示超过原图片大小，可小不可大
  3. color：图片颜色
  4. colorBlendMode：图层混合
  5. repeat：重复模式，noRepeat，reapeat，repeatX和repeatY
  6. alignment：控制图片的摆放位置
  7. centerSlice：当图片需要被拉伸时显示，centerSlice定义的矩形区域（9个点）被拉伸
  8. matchTextDirection：与Directionality配合使用。TextDirection.ltr从左向右展示图片，TextDirection.rtl从右向左展示图片
  9. image：抽象类，需要自己实现获取图片数据的操作
  10. gaplessPlayback：当ImageProvide发生变化时，重新加载图片过程中，原图片的展示是否保留。true保留，false不保留，直接等待下一张图片加载
