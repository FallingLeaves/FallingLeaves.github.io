宽高尺寸处理

SizedBox(设置具体尺寸)

SizedBox组件是一个特定大小的盒子，这个组件强制他的child有特定的宽度和高度，如果宽度或高度为null，则此组件将调整自身大小以匹配该维度中child的大小。

SizedBox主要属性

属性名 类型 说明
width double 宽度值
height double 高度值

ConstrainedBox(限定最大最小宽高布局)

ConstrainedBox的作用是限定子元素child的最大宽度、最大高度、最小宽度和最小高度。

ConstrainedBox主要属性

属性名 类型 说明
constraints BoxConstraints 添加到child上的额外限制条件
child Widget 子元素

LimitedBox(限定最大宽高布局)

LimitedBox组件是限制类型的组件，可对最大宽高进行限制，LimitedBox是将child限制在其设定的最大宽高中，但这个限定是有条件的。当LimitedBox最大宽度不受限制时，child的宽度就会受到这个最大宽度的限制，高度同理。

LimitedBox主要属性

属性名 类型 说明
maxWidth double 限定的最大宽度，默认double.infinity
maxHeight double 限定的最大高度，默认double.infinity

AspectRatio(调整宽高比)

AspectRatio的作用是根据设置调整子元素child的宽高比。AspectRatio的布局行为分为两种情况：

  • AspectRatio首先会在布局限制条件允许的范围内尽可能的扩展，Widget的高度是由宽度和比率决定的，按照固定比率去尽量占满区域。
  • 如果在满足所有限制条件后无法找到可行的尺寸，AspectRatio最终将会优先适应布局限制条件，而忽略设置的比率。

AspectRatio主要属性

属性名 类型 说明
AspectRatio double 宽高比，最终可能不会根据这个值去布局，具体看综合因素，外层是否允许按照这种比率进行布局，只是一个参考值。
child Widget 子元素

FractionallySizedBox(百分比布局)

FractionallySizedBox组件会根据现有空间来调整child的尺寸，所以就算为child设置了具体的尺寸，也不起作用。

FractionallySizedBox的布局主要跟他的宽高因子(两个参数)有关，当参数为null或设置具体参数时，布局表现不一样。还有一个辅助参数alignment，作为对齐方式进行布局。具体有以下两种情况：

  • 设置了具体的宽高因子，具体的宽高根据现有空间宽高 * 因子，当宽高因子大于1的时候，可能会超出父组件范围
  • 没有设置宽高因子，则填满可用区域

FractionallySizedBox主要属性

属性名 类型 说明
alignment AlignmentGeometry 对齐方式，不能为null
widthFactor double 宽度因子
heightFactor double 高度因子
