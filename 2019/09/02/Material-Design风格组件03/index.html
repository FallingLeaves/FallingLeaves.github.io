TextField(文本框组件)

TextField组件是用来做文本输入的组件，而Text组件主要用于显示文本，并不接受输入文本。

TextField组件属性

属性名 类型 说明
maxLength int 最大长度
maxLine int 最大行数
autocorrect bool 是否自动更正
autofocus bool 是否自动对焦
obscureText bool 是否是密码
textAlign TextAlign 文本对齐方式
style TextStyle 输入文本的样式
inputFormatters List<TextInputFormatter> 允许的输入格式
onChanged ValueChanged<String> 内容改变时的回调
onSubmitted ValueChanged<String> 内容提交时的回调
enabled bool 是否禁用

Card(卡片组件)

Card组件，内容可以由大多数类型的Widget构成，但通常与TistTile一起使用。Card有一个child，但可以是支持多个child的列、行、列表、网格或其他小部件。默认情况下，Card将其大小缩小为0像素。可以使用SizeBox来限制Card的大小。

Card组件属性

属性名 类型 默认值 说明
child Widget 组件的子元素
margin EdgeInsetsGeometry 围绕在decoration和child之外的空白区域，不属于内容区域
shape ShapeBorder RoundedRectangleBorder Card的阴影效果，默认为圆角的长方形。弧度为4.0.
