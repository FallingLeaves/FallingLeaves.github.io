flutter常用组件–列表组件

在flutter中，用ListView显示列表，支持垂直和水平，可以分为：

  • 水平的列表
  • 垂直的列表
  • 数据量非常大的列表
  • 矩阵式的列表

基础列表组件ListView

ListView属性

属性名 类型 默认值 说明
scrollDirection Axis Axis.vertical 列表的排列方式，Axis.vertical为垂直方式，Axis.horizontal为水平方式
padding EdgeInsertsGeorge 列表内部的空白区域
reverse bool false 列表排列反向
children List<Widget> 列表元素

水平列表

把ListView组件的scrollDirection属性设置为Axis.horizontal即可。

长列表组件

长列表组件也是使用ListView作为基础组件，需要添加一个列表项构造器itemBuilder。

网格列表组件GridView

使用GridView创建网格列表组件的方式：

  • GridView.count通过单行展示个数创建GridView
  • GridView.extent通过最大宽度创建GridView

GridView属性

属性名 类型 默认值 说明
scrollDirection Axis Axis.vertical 滚动方向
reverse bool false 默认是从上或从左向下或向右滚动，这个属性控制是否反向
controller ScrollController 控制child滚动时的位置
primary bool 是否与父节点的PrimaryScrollController所关联的主滚动视图
physics ScrollPhysics 滚动的视图是如何响应用户的输入
shrinkWrap bool false 滚动方向的滚动视图内容是否应该由正在查看的内容所决定
padding EdgeInsetsGeometry 四周空白区域
GridDelegate SliverGridDelegate 控制GridView中节点布局的delegate
CacheExtent double 缓存区域
