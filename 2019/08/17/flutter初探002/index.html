flutter常用组件–图标及按钮组件

图标组件Icon

展示图标的组件，该组件不可交互，要实现交互的图标，可以考虑使用IconButton组件。图标相关的组件有：

  1. IconButton：可交互的Icon
  2. Icons：框架自带Icon集合
  3. IconTheme：Icon主题
  4. ImageIcon：通过AssetImages或者其他图片显示Icon

Icon常见属性

属性名 类型 默认值 说明
color Color null 图标的颜色
icon IconData null 展示的具体图标，可以使用Icons图标列表中的任意一个
style TextStyle null 文本样式
size Double 24.0 图标大小
TextDirection TextDirection TextDirection.ltr Icon组件中也可以添加文本内容

图标按钮组件IconButton

图标组件（IconButton）是基于Material Design风格的组件，可以响应按下事件，并且带有一个水波纹的效果。如果onPress回调函数为null，按钮处于禁用状态，并且不可按下。

IconButton常见属性

属性名 类型 默认值 说明
alignment AlignmentGeometry Alignment.center 定义IconButton的Icon对齐方式，默认居中。Alignment是可以设置x，y的偏移量
icon Widget null 展示的具体图标
color Color null 图标组件的颜色
disabledColor Color ThemeData.disabledColor 图标禁用颜色
iconSize double 24.0 图标大小
onPress VoidCallback null 按钮按下时触发此回调事件
tooltip String ‘’ 按钮按下时的组件提示语句

凸起按钮组件RaisedButton

凸起按钮组件RaisedButton是Material Design中的button，一个凸起的材质矩形按钮，可以响应按下事件，会带有一个触摸的效果

RaisedButton常见属性

属性名 类型 默认值 说明
color Color null 组件的颜色
disabledColor Color ThemeData.disabledColor 组件禁用的颜色
onPress ViodCallback null 按下时的回调事件
child Widget 按钮的child通常为一个Text组件，显示按钮文本
enable bool true 按钮是否为禁用状态
